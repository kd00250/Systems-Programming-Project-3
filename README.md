# Systems-Programming-Project-3
Client - Server Project (both projects must be ran on the same machine)

#Server Project Run Instructions
-Unzip the file
-Keep the Server project inside the folder that contained both projects
-open the folder on vs code or whatever ide you want 
-For vs code you need the following extensions: extension pack for java (by microsoft), debugger for java extensions (by microsoft), language support for java (by red had)
-Run the main file first (before the client)
-See that the server is listening for messages
-See the outputted results in the data folder and displayed in the terminal once recieved from the client

#Client Project Run Instructions 
-Unzip the file
-Move the file that contains the client project out of the folder so that it is by itself (move the folder titled project3 (project3 is the client project) out on the desktop so that it is by itself)
-Open the folder on v code or whatever ide you want 
-For vs code you need the following extensions: extension pack for java (by microsoft), debugger for java extensions (by microsoft), language support for java (by red had)
-Run the main file after you have already started the server first
-Type in what branch you want to send to the server (must be in all caps e.g AUSTX) and enter
-The file will be sent to the server and confirming output will be displayed in the terminal when the transfer is successful and when the client is connected to the server

#Tests for the client instructions
-Once you have the client project open in vs code go to the test folder and click on the testLib file to view the code
-Once you are viewing the code find the same file in the explorer to the left and right-click it 
-Click run tests toward the bottom of the pop-up 
-See the results in "test results" tab next to the terminal
-P.S. vs might have to load java or the java debugger so you might have to wait a minute before running the tests
